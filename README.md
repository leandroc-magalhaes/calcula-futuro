# Calcula Futuro

Site de calculadoras financeiras gratuitas.

## Ferramentas disponíveis

- Calculadora de Juros Compostos

## Objetivo

Ajudar pessoas a simularem investimentos, aportes mensais e crescimento financeiro ao longo do tempo.

## Aviso

As ferramentas possuem finalidade educativa e não representam recomendação de investimento.
